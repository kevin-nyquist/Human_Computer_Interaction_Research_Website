Prototypes

User Story 1

As a student with a busy schedule, I want to be able to save the progress I have made on my assignment so that I can go back and complete assignments on my own timetable.

User Story 2

As a student in a classroom where my professor just reviewed the details of an upcoming assignment on Rampant Strategy and then assigned an in-class poll, I want to be able to conveniently switch between my assignments tab (PeerDoc) and polling tab (PeerPoll) so that I don\'t waste class time being confused by the navigation.

User Story 4

As a student using Rampant Strategy for a class, I keep having to open the app and navigate to assignment feedback to see if its been released I want to easily be able to see my assignment feedback so that I can do better on the next assignment.

User Story 5

As a student who is on the brink of passing the class, I want to be able to see all of my upcoming assignment deadlines in one place so that I am able to plan my workweek accordingly and not procrastinate.
